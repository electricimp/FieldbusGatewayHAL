# Fieldbus Gateway Hardware Abstraction Layer

Use the FieldbusGateway_005 tables to access the hardware pins, uart, spi, and i2c bus objects.

## FieldbusGateway_005

* LED_RED
* LED_GREEN
* LED_YELLOW
* MIKROBUS_AN
* MIKROBUS_RESET
* MIKROBUS_SPI
* MIKROBUS_CS
* MIKROBUS_PWM
* MIKROBUS_INT
* MIKROBUS_UART
* MIKROBUS_I2C
* XBEE_RESET
* XBEE_AND_RS232_UART
* XBEE_DTR_SLEEP
* RS485_UART
* RS485_nRE
* WIZNET_SPI
* WIZNET_RESET
* WIZNET_INT
* USB_EN
* USB_LOAD_FLAG
